# Market GG Gateway

Market GG 게이트웨이는 각 서비스 또는 데이터와 접속하고 API 호출을 위한 관리, 모니터링, 인증 및 일반 엑세스 제어를 담당하는 API 관리 시스템입니다.

## Getting Started

```bash
./mvnw spring-boot:run
```
